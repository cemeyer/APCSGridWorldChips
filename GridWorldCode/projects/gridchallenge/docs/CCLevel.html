public class CCLevel

The CCLevel class represents Chip's Challenge levels. You probably don't want to actually use this class, but instead use CCLevelSet.

Constructor(s)
Definition: Notes:
public CCLevel(byte[] leveldata) Here leveldata is an array of all the bytes in the levelset file that are part of this level.

Getter(s)
Definition: Notes:
public int getObjectAt(int x, int y, int layer) Return the integer object id of the object residing at the desired location.
public LinkedList<Point> getMonsterLocations() Returns a linked list of the monsters that move on this level.
public int getLevelNumber() Returns which level this is.
public int getTimeLimit() Returns how many seconds the player has for this level.
public int getChipCount() Returns the number of chips needed to pass the exit lock on this level
public CCLevel getNextLevel() Returns the next level in this levelset.
public String getTitle() Returns the title of the level.
public String getHint() Returns the hint for this level.
public String getPassword() Returns this level's password.
public Point getTrapForButton(Point button) Returns the trap associated with this button.
public Point getCloneForButton(Point button) Returns the cloning machine associated with this button.

Setter(s)
Definition: Notes:
public void setNextLevel(CCLevel next) Used by CCLevelSet only. You *could* use this to change the order of levels or omit them, but shouldn't.
